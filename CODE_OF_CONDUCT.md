# Code of Conduct

## Our Commitment

我们致力于为所有贡献者提供友好、安全、尊重的协作环境。

## Expected Behavior

- 尊重不同背景与观点。
- 接受建设性反馈，并保持专业沟通。
- 关注问题本身，避免人身攻击。
- 在公开讨论中保持礼貌与包容。

## Unacceptable Behavior

- 任何形式的骚扰、歧视、侮辱或威胁。
- 恶意引战、刷屏、钓鱼或人身攻击。
- 未经允许公开他人隐私信息。

## Enforcement

维护者有权删除不当内容，并对违规账号采取警告、临时或永久禁言等措施。

## Scope

本准则适用于仓库内外与本项目相关的交流场景（Issue、PR、Discussion、社区群等）。
